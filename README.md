# Archita Biswal Portfolio

A React + Vite portfolio site for Archita Biswal.

## Features

- Animated hero typing effect
- Skills tabs with progress bars
- Filterable project cards
- Education, certifications, experience, and contact sections
- Dark/light theme toggle

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To Netlify

This project is ready for Netlify with the included `netlify.toml`.

1. Push the repository to GitHub.
2. In Netlify, create a new site from Git.
3. Select the repository and use the default settings.
4. Netlify will run `npm run build` and publish the `dist` folder.

## Notes

- The contact form is currently a front-end demo and does not submit to a backend.
- Update the social links, email, and phone number as needed before publishing.
